# 一个月学会 Kubernetes

![header](./header.png)

哈喽，欢迎来到我的课程。我希望本课程可以给大家带来良好的学习体验。

在每一章中都有一个明确的重点，一个有用的话题，并且这些话题是相互关联的，让你有一个全面的了解，了解如何在实践中使用 Kubernetes。你需要大量的练习，每天练习巩固每一章获得的知识，形成肌肉记忆。

可以移步到 [GitHub Pages](https://yyong-brs.github.io/learn-kubernetes/) 页面进行阅读。
